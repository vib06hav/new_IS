## Key Insights

1. **Google Gemini models dominate budget tier** — Flash Lite (₹0.42) and 2.0 Flash (₹0.42) are 5–10x cheaper than Claude Opus while handling 1M context.

2. **DeepSeek V3.1 Terminus is sweet spot for extraction** — ₹0.75/PDF, deterministic routing, no fallback pool risk.

3. **Claude Opus 4.6 justifies premium cost** — 1M context + instruction reliability. Worth it if JSON structure or complex logic is critical.

4. **GPT-4o is Goldilocks solution** — ₹2.65/PDF. Strong instruction following, reasonable cost, well-documented.

5. **OpenAI GPT-4o Mini is dangerous** — Ultra-cheap (₹0.38/PDF) but weak at complex instruction following. Not recommended for structured extraction with 40K+ context.

6. **Google Gemma 2/3 are novelty models** — Extremely cheap but very small (4B–27B). Only use if your PDF extraction is trivial.

7. **Reasoning models (R1) are overhead here** — You're not doing math/coding. Standard models (V3, Opus) are sufficient. Skip unless gate validation is failing.
